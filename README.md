Валидатор пароля.

Создайте класс 'Validator'. У него должен быть метод 'password()'.

Метод 'password()' должен возвращать 'isValid(value)'.

Пароль проходит, если:

-это строка

-в нём минимум 8 символов

-в нём есть хотя бы одна цифра

-в нём есть хотя бы одна маленькая латинская буква

-в нём есть хотя бы одна большая латинская буква

Пароль также должен поддерживать методы:

-minLength(length) — меняет минимальную длину пароля

-contains(value)— требует, чтобы в пароле был конкретный символ или строка

Пример:
const v = new Validator();

const schema = v.password();

schema.isValid('Hello123'); // true

schema.isValid('hello123'); // false
